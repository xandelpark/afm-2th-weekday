# 데일리테니스 자세분석 앱 (Daily Tennis Coach) — MISSION

> 매장 고객의 볼머신 연습 영상을 분석해 6축 능력치 + 슬로모션 리플레이로 피드백을 주는 매장 PC 기반 앱

## 1. 한 줄 미션

**"데일리테니스 매장 고객이 볼머신 연습 영상을 매장 PC에 업로드하면, 30초 안의 모든 스윙(포핸드/백핸드)을 자동 분석해 위닝일레븐식 육각형 능력치 + 슬로모션 리플레이를 제공하고, 결과 영상을 SNS로 바로 가져갈 수 있게 해주는 셀프 코칭 + 매장 콘텐츠 자산 앱"**

## 2. 배경 & 동기

- **운영자**: psk1397@gmail.com (데일리테니스 매장 운영자)
- **매장 페인포인트**:
  1. 다른 테니스 매장과의 차별화 콘텐츠가 부족
  2. 매장에서 자체 SNS 운영용 영상 콘텐츠가 필요
  3. 코치 없이 혼자 볼머신 연습하는 초보 고객들이 "내가 잘 치고 있는 건지" 셀프 피드백 받을 방법이 없음
- **핵심 통찰**: 테니스 자세는 "정답"이 없음 → 시중 앱(SwingVision 등)이 흉내내지 못하는 **데일리테니스 자체 룰북**이 곧 브랜드 자산

## 3. 타겟 사용자

- **(주)** 매장 볼머신을 빌려 혼자 연습하는 일반 고객 (코치 없이 셀프 피드백 필요)
- **(주)** 데일리테니스 레슨 회원 (코치가 옆에서 분석 도구로 같이 활용)
- 둘 다 핵심 사용자

## 4. 핵심 사용자 시나리오

```
[매장 도착] 고객이 볼머신 코트 입장
   ↓
[연습] 약 4분 정도 볼머신 연습 (포핸드/백핸드 혼합)
   ↓
[자기 폰으로 셀프 촬영] 측면 각도로 촬영
   ↓
[폰에서 직접 편집] 잘 친 부분만 30초 이내로 잘라냄
   ↓
[매장 PC 앞에 앉음] PC 화면에 떠있는 QR 코드 스캔
   ↓
[모바일 업로드 페이지 → 영상 전송] PC로 자동 도착
   ↓
[자동 분석] 30초 영상 안 모든 스윙 자동 분리 + 포/백 자동 분류 + 6축 채점
   ↓
[결과 화면] 통합 점수(0~100) + 육각형(5단계 별점) + 슬로모션 리플레이
   ↓
[SNS 공유] 결과 영상 다운로드 (또는 QR로 폰에 받기)
   ↓
[저장 시 가입 유도] 비회원도 분석은 가능, 결과 저장하려면 그때 가입 → 히스토리 누적
```

## 5. 운영 모델 (확정)

- **하이브리드 모델 (C)**: 매장 PC에서만 촬영/분석, 결과·점수·영상은 개인 계정에 누적
- **비회원 분석 OK**: 분석은 누구나 → 결과 저장 시점에 가입 유도 (전환율 최적화)
- **회원 가입 후**: 내 스윙 히스토리 누적, 한 달 전 대비 능력치 변화 추적
- **영상 업로드**: QR 코드 (PC가 QR 띄우면 폰으로 스캔 → 모바일 업로드 페이지 → PC 전송)
- **영상 길이 제한**: 30초 이내 (사용자가 폰에서 직접 잘 친 부분만 잘라 업로드)
- **손잡이 처리**: 첫 업로드 직전 또는 회원가입 시 "오른손/왼손" 1회 선택

## 6. MVP 스코프

### 포함 (V1)
- 포핸드 + 백핸드 자동 분류 분석 (혼합 영상 OK)
- 30초 영상 안 여러 스윙 자동 분리 (손목 속도 피크 검출 기반)
- 6축 통합 점수 + 슬로모션 리플레이 (결과 화면 1장으로 통합 - **A 통합형**)
- QR 업로드 페이지
- 회원가입/로그인 + 손잡이 설정
- 결과 영상 다운로드 (SNS용)

### 제외 (V2 이후)
- 서브, 발리 분석
- 임팩트 정확도 ("공을 라켓 어디에 맞췄나" — 공 추적 필요)
- 한 달 전/지난주 대비 변화 그래프
- 매장 코치 코멘트 첨부 기능
- 자동 임팩트 검출 (UX 우회로 해결됨 — 사용자 직접 30초 컷)
- 스윙별 카드형 상세 뷰 (B 카드형 — V2에 추가 검토)

## 7. 육각형 6축 정의 (조합 A — 시간순 흐름형)

스윙의 시간 흐름(준비→백스윙→임팩트→팔로우스루)을 따라가며 매 단계를 평가하여, 고객이 자기 약점을 시간순으로 이해할 수 있도록 함.

| # | 축 이름 | 측정 방법 | 판단 근거 |
|---|---------|-----------|-----------|
| 1 | **준비 자세** | 백스윙 시작 시 어깨/무릎/스탠스 각도 | 측면 키포인트 각도 |
| 2 | **백스윙 안정성** | 백스윙 정점에서 라켓팔 위치 일관성 + 어깨 회전 각도 | 백스윙 정점 키포인트 분포 |
| 3 | **체중 이동** | 백스윙→임팩트 사이 골반 X좌표 이동 폭 | 골반 키포인트 시계열 (논문 다수) |
| 4 | **회전 타이밍 (운동사슬)** | 골반 회전 → 어깨 회전 → 팔 휘두름 시간차 | 골반·어깨 회전 정점 시간차 (Elliott 논문 등) |
| 5 | **임팩트 자세** | 임팩트 추정 순간(손목 속도 정점)의 무릎 굴곡, 척추 직립도, 머리 위치 | 임팩트 시점 다관절 각도 |
| 6 | **팔로우스루** | 임팩트 후 팔 궤적 + 종료 위치, 어깨 위로 마무리 여부 | 임팩트 이후 팔 키포인트 궤적 |

### 룰북 / 채점 기준 운영 방침

- **MVP 출시 시점**: 매장 자체 코칭 자료/논문 없음 → 일반론 기반 임의 가중치로 시작
- **출시 후**: 실제 고객 데이터 보며 매장 코치들과 함께 가중치 보정 (운영 회고 루프)
- **포핸드/백핸드 룰북 분리 운영**: 두 종목 채점 기준이 다르므로 별도 가중치 테이블 관리

## 8. 점수 표시법 (하이브리드)

- **종합 점수**: 0 ~ 100점 (SNS 친화적, "나 78점 받음!")
- **6축 각각**: 1 ~ 5단계 별점 (정밀도 압박 ↓, 부드러운 인상)
- **포핸드 평균 / 백핸드 평균**: 각각 100점 기준으로 별도 표시 (혼재 영상에서 종목별 약점 식별 가능)

## 9. 성공 지표 (운영 KPI)

### 단기 (출시 후 1~3개월)
- **사용 빈도**: 매장 방문 고객 중 분석기 사용 비율 ≥ 30%
- **가입 전환율**: 비회원 분석 → 결과 저장 시 회원가입 비율 ≥ 50%
- **SNS 콘텐츠 생산**: 데일리테니스 인스타에 분석 영상 주 2회 이상 업로드 가능

### 중기 (3~6개월)
- **재방문 동기**: 회원 평균 재방문 주기 단축 (앱 사용 회원 vs 미사용 회원 비교)
- **레슨 전환**: 일반 고객 → 레슨 회원 전환율 상승
- **브랜드 인지**: 분석 영상 보고 신규 고객이 매장 방문 (유입 채널로 SNS 클릭 측정)

### 장기 (6개월~)
- **데일리테니스 룰북**의 매장 자산화 (다른 매장과 차별화되는 채점 기준)
- 회원 스윙 히스토리 누적 → 락인 효과

## 10. 차별점 (vs 시중 앱)

| 항목 | 시중 앱 (SwingVision 등) | 데일리테니스 |
|------|-------------------------|--------------|
| 채점 기준 | AI 블랙박스, 글로벌 일반론 | 매장 자체 룰북 (운영 중 보정) |
| 사용 환경 | 개인 폰 앱, 사용자 부담 큼 | 매장 PC + QR (매장 와서만 가능) |
| 콘텐츠 | 개인 기록용 | **즉석 SNS 콘텐츠** + 매장 운영 자산 |
| 락인 | 앱 자체에 종속 | 데일리테니스 매장 + 코치 + 룰북 종속 |
| 비용 | 월 구독 | 매장 회원 무료/매장 부가 서비스 |

## 11. 기술 제약 메모

- **포즈 추정**: MediaPipe BlazePose / TF.js MoveNet 사용. 사람 1명, 측면 촬영 기준 best
- **측정 가능**: 관절 각도, 회전 타이밍, 무게중심 이동, 손목 속도, 머리 흔들림
- **측정 불가 (V2 이후)**: 라켓 자체 추적, 공 추적, 임팩트 정확도
- **임팩트 시점 추정**: 손목 속도 정점으로 근사
- **포/백 자동 분류**: 라켓 든 손 좌→우/우→좌 + 어깨 회전 방향 (오/왼손 정보 활용, 95%+ 정확도 기대)
