# 🎯 APP MISSION

## Project Name
**Marian Wedding — Thank You Card** (운영 슬기 · 마리안 웨딩)

## Mission Statement
마리안 웨딩이 신랑·신부에게 ID/PW를 발급해 드리면, 신랑·신부가 본식 스냅 한 장을 올렸을 때 "수정 없이 바로 보낼 수 있는" 완성도의 결혼식 땡큐카드를 3가지 톤(미니멀·격식·감성)으로 자동 생성해 주는 서비스.

## Problem Statement
- 결혼식 후 신랑·신부는 하객들에게 감사 카드를 보내고 싶어하지만, 본식 스냅을 받은 뒤 직접 캔바·미리캔버스로 만드는 건 폰트·여백·카피 감각이 어려워 "사진 그냥 보내기"로 끝나는 경우가 많다.
- 마리안 웨딩 입장에서도 매 부부마다 수동으로 카드를 디자인해 주기엔 작업량이 크다.
- 본식 스냅 + 적절한 자리에 놓인 한두 줄의 타이포그래피 + "thank you" 컬리그래피 — 이 "적절함"을 매번 사람이 잡고 있다.

## Target Users
- **1차 (관리자)**: 마리안 웨딩 운영자(슬기) — 새 부부가 식을 마치면 ID/PW를 발급해서 카드 도구를 열어 줌. 사용 횟수·만료일 관리.
- **2차 (일반 사용자)**: 마리안에서 결혼식을 한 신랑·신부 — 발급받은 계정으로 로그인해 본식 스냅을 올리고, 자동 생성된 3가지 카드 중 마음에 드는 걸 다운로드해 하객에게 전송.

## App Type
- [x] Public/Commercial (관리자 1 + 다수 고객 구조, v1은 단일 스튜디오)
- [ ] Personal/Private Use

## Core Features (v1)
1. **관리자 승인형 회원가입** — 마리안 운영자가 부부 단위로 ID/PW를 직접 발급하고 활성/비활성·만료일·할당량을 토글. 일반 회원이 자유롭게 가입할 수 없음.
2. **본식 스냅 1장 업로드 + 부부 정보 슬롯** — 결혼식 당일 촬영된 본식 스냅 1장 업로드 + 신부/신랑 이름·식 일자·장소·감사 메시지 슬롯 입력. 카테고리 선택은 없음(v1은 땡큐카드 단일).
3. **자동 땡큐카드 합성 (3 변형 동시 생성)** — 같은 사진·슬롯으로부터 3가지 톤을 동시에 출력:
   - *미니멀 땡큐* (러블리 미니멀, 인스타 피드용): 사진 72% + 누드 핑크 캡션 띠 28%, 손글씨 `thank you` + 부부·날짜 한 줄
   - *격식형 땡큐* (청첩장 톤의 정중한 감사장): 풀블리드 사진 + 모노그램 + 큰 손글씨 `Thank you` + 한국어 감사 + 일자/장소/이름
   - *감성 땡큐* (한국어 종서 감성): 풀블리드 사진 + 우측 한국어 종서 + 좌하 `Starring · 부부명 · — Fin —`
4. **부분 수정 에디터 (캔바 라이트)** — 카드 안 텍스트 블록을 클릭으로 선택, 드래그 이동, 우하단 핸들로 크기 조절. 카드 밑 인라인 컨트롤로 사진 X/Y/Zoom + 텍스트 X/Y/Scale + **글씨 문구 직접 수정**. 자석 스냅(카드 중심·가장자리·다른 블록 정렬). *(목표: 첫 결과의 완성도가 높아 손댈 일이 거의 없게)*
5. **결과물 다운로드** — PNG 1080×1350(인스타) / 1080×1080(스퀘어) 두 사이즈로 저장.

## Anti-Scope (v1에서 명시적으로 하지 않을 것)
- ❌ AI 이미지 생성·변형 (사진은 원본 그대로 사용)
- ❌ **본식 스냅(땡큐카드) 외 다른 카테고리** — 야외 스냅 / 제품 사진 / 풍경 사진은 v1에서 제외 (확장 시 별도 정의)
- ❌ 결제·구독·플랜 (관리자 수동 발급으로만)
- ❌ 자유 가입·소셜 로그인 (관리자 발급 ID/PW만)
- ❌ 캔바급 자유 편집(요소 추가·도형·필터·그리드 등) — 텍스트 위치/크기/문구 수정만
- ❌ 다중 스튜디오/멀티 테넌시 — v1은 **마리안 웨딩 단일 스튜디오** 전용
- ❌ 협업·댓글·공유 권한 관리
- ❌ 모바일 네이티브 앱 (웹 반응형으로 충분)

## Success Metrics
| Metric | Target | Timeframe |
|---|---|---|
| 본식 스냅 1장 업로드 → 3 카드 결과물 받기까지 소요 시간 | < 8초 | v1 출시 시점 |
| 받은 카드를 **수정 없이** 그대로 다운로드하는 비율 | ≥ 70% | 출시 후 1개월 |
| 마리안 부부 1팀당 평균 다운로드 카드 수 | ≥ 2장 | 출시 후 1개월 |
| 관리자 계정 발급 → 부부의 첫 카드 다운로드까지 이탈률 | ≤ 20% | 출시 후 1개월 |
| 마리안 한 달 신규 부부 중 본 서비스 사용률 | ≥ 60% | 출시 후 3개월 |

## Open Questions
- **부부 슬롯 입력 시점**: 신부·신랑 이름·식 일자·장소·감사 메시지를 (a) 업로드 시 한 번 입력 vs (b) 결과 카드 인라인 편집. v1 디폴트는 (a) + 카드 안 인라인 수정도 허용.
- **카드 디폴트 카피**: 한국어 감사 메시지·영문 sub의 디폴트 풀을 마리안 톤(러블리 + 정중)으로 5~8종 마련하고 부부에게 무작위 매칭 vs 한 가지 고정. v1 = 한 가지 고정 + 부부가 직접 수정.
- **마리안 로고 워터마크**: 우하단에 마리안 로고 PNG 자동 합성을 디폴트 ON 또는 OFF? 디폴트 OFF, 관리자 토글.
- **부부에게 노출되는 무드**: v1은 미니멀(인스타) · 격식(인쇄·문자 첨부용) · 감성(영화 톤) 3종. 추후 부부가 더 보고 싶어 하는 톤이 나오면 추가.
- 폰트 라이센스 (Pretendard · Noto Serif KR · Cormorant Garamond · Allura · Playfair Display) — 모두 OFL/SIL, v1 OK.
- 만료일·사용 횟수 디폴트: 발급 후 30일 / 50장 (마리안 운영자가 변경 가능).
