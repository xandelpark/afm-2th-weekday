# 아기-부모 카툰 쇼츠 메이커 (Baby Cartoon Shorts Maker)

## 한 줄 요약
부모가 아기와의 일상 대화를 음성/영상으로 업로드하면, 발화자별로 카툰 캐릭터와 자막이 입혀진 짧은 카툰 영상을 만들어 SNS에 바로 공유할 수 있게 해주는 도구.

## 누구를 위한 것인가 (페르소나)
- 어린 자녀를 둔 부모
- 아기와의 귀여운 대화 순간을 가족·SNS에 공유하고 싶음
- 영상 편집 기술 없음, 5분 안에 끝내고 싶음
- **앱 제작자 본인이 1차 사용자** ("내가 늘 필요했던 것을 만든다")

## 사용자 워크플로우 (사용자가 직접 설계한 흐름)
1. 아기와의 대화 영상/음성 파일 업로드 (mp4 / m4a / mp3)
2. 시스템: 발화자를 **사람1 / 사람2 / 사람3** 익명으로 자동 구분 + 부모 발화 자동 transcription
3. 사용자: 화면에서 "사람1 = 아빠", "사람2 = 아기", "사람3 = 엄마" 라벨링
4. 사용자: 각 역할에 카툰 캐릭터 선택 (아빠 5종 / 엄마 5종 / 아기 5종 중 각 1개)
5. 아기 segment 자막은 의성어 풀("응애!", "꺄르륵", "옹알옹알" 등)에서 자동 배치 (사용자 편집 가능)
6. 미리보기 확인 후 MP4 생성 → 다운로드 + SNS 공유

## 무엇을 (앱이 하는 일)
음성/영상 입력을 받아 발화자 구분 → 사용자가 캐릭터·라벨 매핑 → 카툰 캐릭터·자막·원본 음성을 합성한 MP4 출력. 출력은 다운로드 또는 인스타·유튜브로 공유.

## 왜 (해결하는 욕구)
- 아기와의 귀여운 대화는 일반 영상으로 남기기엔 평범 — 카툰화하면 보존 가치와 공유 가치 상승
- 유튜브 쇼츠/릴스에서 비슷한 카툰 콘텐츠가 꾸준히 바이럴 — 검증된 시청자 욕구
- 영상 편집 못 하는 부모도 5분 만에 만들 수 있어야 함

## 핵심 가치 (다른 자동 자막앱과 무엇이 다른가)
- 단순 자막이 아닌 **카툰 캐릭터 + 발화자별 시각 표현**
- 자동 발화자 구분 + 사용자 라벨링의 영리한 결합 (자동화 어려운 부분만 사용자에게)
- 아기 옹알이를 의성어로 귀엽게 표현
- 처리 후 1시간 내 자동 삭제 (개인정보 보호)
- 결과물은 MP4 다운로드 + 인스타 공유 + 유튜브 다이렉트 업로드 (한 곳에서 끝)

## MVP 성공 정의 (측정 가능)
- 본인이 본인 아기 영상 1개로 처음부터 끝까지 1번 성공 — 업로드 → 라벨링 → 캐릭터 선택 → MP4 다운로드까지 막힘 없음
- MP4 출력의 음성·자막 동기화 오차 ±1초 이내
- 발화자 자동 구분 정확도 ≥ 80% (사람1/2/3 단위 — 누구인지가 아니라 "다른 사람으로 구분되는가"만 평가)
- 처리 시간 ≤ 입력 영상 길이 × 3 (1분 영상 → 3분 안에 출력)
- 업로드된 원본 파일이 1시간 후 자동 삭제 확인됨
- 본인이 만든 카툰 영상을 가족 단톡방·SNS에 1회 이상 공유

## 명시적 비범위 (Out of Scope — v2 이후)

## 데이터·프라이버시
- 업로드된 원본 파일은 처리 후 **1시간 보관 → 자동 삭제**
- 화면에 "1시간 후 자동 삭제됨" 명시 표시
- 음성은 발화자 구분/transcription 위해 AssemblyAI 클라우드로 전송됨 (사용자 동의 체크박스)
- DB에는 메타데이터만 (파일명, 업로드 시각, 삭제 예정 시각, 발화자 라벨, 캐릭터 매핑)

## 기술 스택 메모
- Backend: Node.js + Express, 단일 `server.js`
- DB: Supabase Postgres (Week-5 공유 프로젝트 재사용 + `baby_cartoon` 스키마)
- Storage: Supabase Storage 또는 로컬 `./uploads/` (1시간 만료 정책)
- 음성 인식 + 발화자 구분: **AssemblyAI API** (`speaker_labels: true`, 한국어 지원 확인)
- 비디오 합성: **fluent-ffmpeg** (Node.js) — 캐릭터 PNG + 자막 + 원본 음성 → MP4
- Frontend: React + Tailwind CDN, 단일 `index.html` + PWA manifest/sw
- 캐릭터 자산: 무료 일러스트 라이브러리 큐레이션 (5종 × 3역할 = 15장 PNG)
- SNS 공유:
  - 인스타 = Web Share API (모바일 브라우저)
  - 유튜브 = YouTube Data API v3 OAuth (Day 6-7 stretch goal)
  - MP4 다운로드 = 기본 제공
