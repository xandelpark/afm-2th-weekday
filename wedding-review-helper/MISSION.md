# 🎯 APP MISSION

## Project Name
**더화려한날엔 후기비서** (가칭)
— 본식스냅 업체 "더화려한날엔"(thewonderfulday.co.kr) 고객 후기 작성 도우미

## Mission Statement
본식스냅 고객이 후기 이벤트에 참여할 때, 백지 앞에서 막막해하지 않고 몇 가지 안내 질문에 답만 하면 웨딩카페·개인블로그에 바로 올릴 수 있는 자연스러운 후기 초안을 AI가 대신 써주는 무설치 웹 도구.

## Problem Statement
후기 이벤트를 열어도 "뭐라고 써야 할지 모르겠다"는 이유로 고객이 작성을 미루거나 성의 없는 한 줄 후기만 남긴다. 업체 입장에선 웨딩카페·블로그 노출(마케팅 자산)과 사회적 증거를 얻을 기회를 놓친다. 고객은 쓰기 싫은 게 아니라 **시작이 어려운 것**이다.

## Target Users
- **1차 사용자**: "더화려한날엔"과 계약했거나 예식을 마친 신랑·신부 (모바일에서 업체가 보내준 링크로 접속)
- **수혜자**: 업체(더화려한날엔) — 후기 작성률·퀄리티 상승으로 마케팅 효과

## App Type
- [ ] Personal/Private Use
- [x] Public/Commercial (업체가 고객에게 배포하는 도구)

## Core Features (v1)
1. **후기 종류 선택** — 진입 시 "계약후기 / 사진후기" 두 갈래 선택
2. **계약후기 가이드 인터뷰** — "어떻게 알게 됐나 / 다른 업체와 비교 / 상담 인상 / 계약 결정 이유" 등 안내 질문에 짧게 답변
3. **사진후기 가이드 인터뷰** — "예식 당일 경험(분위기·진행) + 맛보기 보정본 5장 감상" 두 축의 안내 질문에 답변 (사진 업로드/편집 없음, 사진은 고객이 직접 첨부)
4. **AI 후기 초안 생성** — 고객의 짧은 답변을 바탕으로 Claude가 웨딩카페·개인블로그 톤의 자연스러운 중장문 후기 글을 생성. 인위적이지 않고 실제 후기처럼
5. **다듬기 & 복사** — 생성된 글을 고객이 직접 수정 가능, 원클릭 복사해서 카페·블로그에 붙여넣기

## Anti-Scope (What this app will NOT do in v1)
- 사진 업로드·보정본 레이아웃·카드 이미지 생성 (사진은 고객이 카페·블로그에서 직접 첨부)
- 후기 제출 수집·업체 관리자 페이지·이벤트 정산 (생성·복사까지만; 고객이 올린 뒤 카카오로 알림)
- 로그인·회원가입·계정 관리 (링크만으로 즉시 사용)
- 네이버 플레이스/인스타 등 단문·해시태그 전용 포맷 (v1은 웨딩카페·블로그 중장문 1종에 집중)

## Success Metrics
| Metric | Target | Timeframe |
|--------|--------|-----------|
| 링크 접속 → 후기 생성 완료율 | ≥ 70% | v1 출시 |
| 후기 1건 작성 소요 시간 | ≤ 5분 | v1 출시 |
| 후기 이벤트 참여(작성)율 (기존 대비) | 2배 이상 | 출시 3개월 |
| 생성 글 그대로 사용/약간 수정 후 사용 비율 | ≥ 80% | 출시 3개월 |

## Open Questions
- AI 문체 톤: 20~30대 신부 말투 반말체 vs 정중한 존댓말체 — 기본값과 선택 옵션 제공 여부
- 계약후기는 계약 직후, 사진후기는 예식 후 — 링크를 하나로 통합할지 시점별로 분리 배포할지
- 안내 질문 문항 수 (너무 많으면 이탈, 너무 적으면 글이 빈약) — 종류별 4~6문항 적정선 튜닝 필요
- Claude API 호출 비용/키 관리 — 실제 구현 방식은 DEV.md(dev-kickstart 단계)에서 결정
